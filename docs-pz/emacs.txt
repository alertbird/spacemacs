
Emacs Shortcuts
===============

C-x o		Switch to other window
C-x 1		Keep one window (the current window)
C-g 		Cancel any command

C-h w		Where is command

C-h c		Describe a command
C-h k		Help on a key bounding

C-h a		Command Apropos. search commands by key words.
C-h m		Help on a mode
C-h ?		Help on help
C-h r		Emacs manual

C-M-v		Scroll another window without moving cursor into the window (Shift - reverse order)

Tips:

Set custom-file
http://stackoverflow.com/questions/5052088/what-is-custom-set-variables-and-faces-in-my-emacs#5058752
Set custom-file. (The custom-file is the file where Emacs saves customizations. If you don’t set a specific file for this, you will end up getting all customizations saved into .emacs which you really don’t want.) The code for this is simple. Just do (setq custom-file “the-file-name.el”)
